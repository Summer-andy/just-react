# FunctionComponent

hooks使FunctionComponent可以获得state，他使用一种与ClassComponent不同的Update。

对应的结构如下：

const update: Update<S, A> = {
  eventTime,
  lane,
  suspenseConfig,
  action,
  eagerReducer: null,
  eagerState: null,

  next: (null: any)
};

Update在dispatchAction方法内创建，你可以从这里看到Update创建的逻辑

可以看到，大部分字段与ClassComponent Update相同。不同的字段意义如下：

  • eagerReducer与eagerState：这两个字段与调用流程优化有关，可以理解为：通过提前计算本次更新的state，如果该state与上次更新的state相同则可以取消本次更新的调度。

  • action：类似ClassComponent Update的payload。对于[state, updateState] = useState()来说，payload为调用updateState传递的参数。
